# ⛽ Gasolinera JSM Ultimate

Simulador de surtidor de gasolina con arquitectura moderna, PWA, testing y CI/CD.

## 🛠 Tecnologías
- Vite • ESM • JavaScript
- PWA • Offline • Installable
- Vitest • Playwright • CI/CD
- a11y • i18n • Dark Mode

## 🚀 Desarrollo
```bash
npm run dev
```

## 📦 Build
```bash
npm run build
```

## 🧪 Testing
```bash
npm run test        # Unit
npm run test:ui     # E2E
```

## 🌐 Despliegue
Automático en GitHub Pages vía CI.
